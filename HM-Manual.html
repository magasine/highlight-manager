1. Introdução

O Highlight Manager é uma ferramenta intuitiva projetada para ajudá-lo a destacar e gerenciar informações importantes diretamente em qualquer página da web. Com ele, você pode facilmente marcar trechos de texto, salvar esses destaques para referência futura e organizá-los para uma experiência de leitura e pesquisa mais eficiente. Diga adeus às anotações perdidas e dê as boas-vindas a um sistema de destaque robusto e personalizável!

Este guia irá cobrir desde a instalação até o uso avançado, garantindo que você aproveite ao máximo todas as funcionalidades do Highlight Manager.

2. Instalação (Como um Bookmarklet/Script de Navegador)

Para adicionar o script ao navegador há uma página de instalação que facilitará o processo de geração e instalação do bookmarklet. Visite-a no seguinte endereço: https://magasine.github.io/highlight-manager/

3. Uso Básico

3.1. Destacando Texto

Para destacar um trecho de texto em qualquer página:

  1. Mantenha a tecla Ctrl (Windows/Linux) ou Cmd (Mac) pressionada.
  2. Com a tecla pressionada, selecione o texto desejado com o mouse.
  3. Ao soltar o botão do mouse (e a tecla Ctrl/Cmd), o texto selecionado será automaticamente destacado com a cor padrão.
Atenção: A seleção de texto deve ser contígua. Se a seleção for muito longa (mais de 10.000 caracteres), um aviso será exibido.

3.2. Acessando o Painel de Destaques

Ao ativar o Highlight Manager, um pequeno painel (geralmente no canto superior direito da tela) aparecerá. Este painel é o coração da ferramenta, onde você pode visualizar e gerenciar todos os seus destaques.

4. Gerenciando Destaques

4.1. Lista de Destaques

Dentro do painel, você verá uma lista de todos os destaques criados. Cada item na lista representa um destaque individual e exibe uma prévia do texto destacado.

  • Nome do Destaque: Geralmente, uma parte do texto destacado. Se for muito longo, será truncado.
  • Remover Destaque (✕): Clique neste botão ao lado de um destaque para removê-lo da lista e da página.
  • Caixa de Seleção (Checkbox): Permite alternar a visibilidade de um destaque específico na página. O estado desta caixa de seleção influencia o status do botão global "Ocultar/Exibir Todos os Destaques".
  • Link do Destaque: Clique no texto do destaque na lista para rolar a página até a localização original do destaque. O destaque piscará brevemente para facilitar a identificação.
  • Copiar Destaque: Clique diretamente sobre o texto destacado na página para copiá-lo para a área de transferência. Uma notificação confirmará a cópia.

4.2. Busca de Destaques

Use a caixa de pesquisa no painel para filtrar seus destaques. Digite qualquer termo na caixa de entrada e a lista será atualizada em tempo real para mostrar apenas os destaques que contêm o termo de busca.

  • Campo de Busca: Search highlights...
  • Botão Limpar Busca (×): Clique neste botão para limpar o campo de busca e exibir todos os destaques novamente.

4.3. Ordenação de Destaques

O botão de ordenação permite organizar seus destaques em ordem alfabética.

  • Botão Ordenar (A-Z ⇅): Clique para alternar entre ordenação ascendente (A-Z ↑) e descendente (Z-A ↓) com base no texto do destaque.

4.4. Ocultar/Exibir Todos os Destaques

Você pode ocultar ou exibir todos os destaques na página de uma vez. O status deste botão (Hide ou Show) reflete a visibilidade geral dos destaques: ele exibirá "Hide" se *todos* os destaques estiverem visíveis, e "Show" se *pelo menos um* destaque estiver oculto. Se você clicar neste botão, o estado de visibilidade de *todos* os destaques e de suas caixas de seleção individuais será atualizado.

  • Botão Ocultar/Exibir: Hide (Ocultar) / Show (Exibir). Clique para alternar a visibilidade global de todos os destaques. Isso não remove os destaques, apenas os torna transparentes. Ao clicar, todas as caixas de seleção individuais serão atualizadas para refletir o novo estado global.

4.5. Mesclar Destaques Visíveis

A nova funcionalidade de mesclagem permite combinar o texto de todos os destaques que estão atualmente visíveis na página em um único bloco de texto, que é então copiado para a área de transferência.

  • Botão Mesclar (Merge): Clique neste botão para iniciar o processo de mesclagem.
  • Definição do Separador: Ao clicar em "Merge", uma caixa de diálogo solicitará que você insira um separador (por exemplo, "---" ou "**") que será usado entre os textos de cada destaque mesclado. Você pode usar o separador padrão ou inserir o seu próprio.
  • Cópia Automática: Após a mesclagem, o texto combinado é automaticamente copiado para a sua área de transferência, e uma notificação confirmará a ação.
  • Ordenação da Mesclagem: Os destaques são mesclados na ordem em que aparecem na página.

4.6. Limpar Todos os Destaques

Para remover todos os destaques de uma vez:

  • Botão Limpar: Clear. Ao clicar, uma confirmação será solicitada. Se você confirmar, todos os destaques serão removidos permanentemente da página e do armazenamento local.

5. Configurações e Personalização

5.1. Posição do Painel

O painel do Highlight Manager pode ser arrastado e solto para qualquer lugar na tela. Basta clicar e segurar na barra de título do painel ("Highlights") e arrastar para a posição desejada. A posição será salva automaticamente para futuras sessões.

5.2. Minimizar/Maximizar Painel

Você pode minimizar o painel para ocupar menos espaço na tela.

  • Botão Minimizar/Maximizar (_): Clique para ocultar ou exibir o conteúdo da lista de destaques, mantendo apenas o cabeçalho visível.

5.3. Temas (Claro/Escuro)

O Highlight Manager se adapta ao tema do seu sistema operacional por padrão. No entanto, você pode alternar manualmente o tema dentro do painel.

  • Detecção Automática: O aplicativo tentará usar o tema claro ou escuro com base nas suas preferências de sistema (prefers-color-scheme).
  • Alternar Tema Manualmente: O botão ☀️ Light / 🌙 Dark no cabeçalho do painel permite alternar o tema manualmente. Esta preferência será salva.

5.4. Feedback do Usuário

Pequenas mensagens de feedback aparecerão brevemente na parte superior da tela para informar sobre ações como "Texto destacado!", "Limite de armazenamento excedido", ou "Gerenciador de destaques fechado".

6. Solução de Problemas Comuns

6.1. Destaques Não Aparecem Após Recarregar a Página

  • Verifique o Bookmarklet/Script: Certifique-se de que você está ativando o Highlight Manager (clicando no bookmarklet ou executando o script no console) após a página ser totalmente carregada.
  • Problemas de Armazenamento Local: O Highlight Manager armazena seus destaques no "Local Storage" do seu navegador. Se você limpar o cache ou os dados do site, os destaques podem ser perdidos. Verifique as configurações de privacidade do seu navegador.
  • Conflitos de Script: Em raras ocasiões, outros scripts na página podem interferir. Tente desativar outras extensões ou scripts para ver se o problema persiste.
  • Conteúdo Dinâmico: Páginas que carregam conteúdo dinamicamente (SPA - Single Page Applications) podem ocasionalmente fazer com que os destaques desapareçam temporariamente. O Highlight Manager tenta restaurá-los, mas pode haver um pequeno atraso.

6.2. Destaques Não Podem Ser Criados

  • Seleção Inválida: Certifique-se de que está selecionando texto contíguo. Selecionar elementos não-texto (imagens, vídeos) ou texto em áreas protegidas pode causar problemas.
  • Seleção Muito Longa: Há um limite de 10.000 caracteres para uma única seleção. Se sua seleção for maior, um aviso será exibido.
  • Permissões do Navegador: Embora seja um bookmarklet, em alguns ambientes restritivos, certifique-se de que scripts JavaScript são permitidos.

6.3. O Painel Desapareceu ou Não Abre

  • Ativação: Lembre-se de que o painel só aparece após você ativar o Highlight Manager (clicar no bookmarklet).
  • Fechamento Acidental: Verifique se você não clicou no botão "X" (Fechar) no cabeçalho do painel. Se sim, basta ativar o bookmarklet novamente.
  • Minimização: O painel pode estar minimizado. Verifique se apenas o cabeçalho está visível e clique no botão "_" para expandir.

6.4. Limite de Armazenamento Excedido

Se você criar um grande número de destaques, o navegador pode atingir seu limite de armazenamento local. O Highlight Manager tentará lidar com isso removendo os destaques mais antigos para liberar espaço. Considere exportar ou limpar destaques que não são mais necessários.
